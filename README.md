# Employee Payroll Management System (Excel)

## Project Overview

This project is an Excel-based Employee Payroll Management System developed to automate payroll calculations, attendance tracking, and salary reporting.

## Features

* Employee Master Database
* Attendance Management
* Payroll Calculation Automation
* HRA, Bonus, PF and Tax Calculations
* Net Salary Calculation
* Department-wise Salary Analysis
* Interactive Dashboard
* Pivot Tables and Charts
* Conditional Formatting

## Tools Used

* Microsoft Excel
* VLOOKUP
* IF Functions
* SUM Functions
* Pivot Tables
* Charts
* Data Validation
* Conditional Formatting

## Dashboard KPIs

* Total Employees
* Total Payroll Expense
* Average Salary
* Highest Salary

## Project Outcome

Developed a payroll management system that improves payroll accuracy, automates calculations, and provides meaningful HR analytics through dashboards.
